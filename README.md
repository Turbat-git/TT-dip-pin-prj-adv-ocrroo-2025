# OCRROO

A Python-based OCR and image processing project using OpenCV and Pillow.

---

## Features
- Real-time image capture and processing  
- Easy-to-run with [uv](https://github.com/astral-sh/uv)  
- Python 3.12+ support  

---

## Requirements
Before running the program, ensure you have:
- **Python 3.12 or later**
- **Git** installed
- **uv** (a fast Python package & environment manager)

---

## Installation

### 1. Install uv
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

---

### 2. Verify uv version
```powershell
uv --version
```

### 3. Clone the repository
```bash
git clone https://github.com/Turbat-git/TT-dip-pin-prj-adv-ocrroo-2025
cd TT-dip-pin-prj-adv-ocrroo-2025
```
---

### 4. Download dependencies
```bash
uv sync
```

---

### 5. Run the program
```bash
uv run python preliminary/library_basics.py
```
